<p> is not an inline tag.

This is, rather, a block element

Because statements in different <p> are printed separately.

This is an Emphasized tag. <em> is an inline tag. Statements in different <em> tag are printed one after the other in an inline fashion. <br> tag is used to print different "inline tag's" statements. It occupies space only upto the statement written in this tag

Semantic tags are the tags that clearly defines the content. Example, <h1> (Used to denote important text), <h6> (used to denote less important text), <nav> (used to show navigation information of a webpage)

While Non-Semantic tags tells nothing about the content. Example, <div> (to group certain statements into one division and apply any needed element onto that statement), <span>
